# Matura - baza zadan C++

Baza zadan, z przygotowan do matury z informatyki.

## Struktura

- `Zadania/` - wszystkie rozwiazywane zadania (kazde zadanie ma swoj folder i pliki)
- `build/` - pliki wynikowe po kompilacji (np. `.exe`)
- `.vscode/tasks.json` (opcjonalnie) - automatyczne komendy build/run w VS Code

## Jak to dziala

1. Wybierz plik `.cpp` z folderu `Zadania/`.
2. Uruchom zadanie VS Code: `C/C++: build active file`.
3. Program kompiluje sie do folderu `build/cpp/`.
4. Uruchom zadanie: `C/C++: run active file`, zeby od razu sprawdzic wynik.
